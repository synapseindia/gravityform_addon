This plugin sends email to the users on rotation basis based on thelimits set for each email address. The limits are based on daily , weekly and monthly.  After thsi plugin is activated a Notification add on menu appears in the setting of the form. There you need to enable the add on by clicking the check box. Email address can be added and there limits will be set for daily, weekly and monthly. Whenever gravity form is submitted the mails will go to the emails defined in the addon. The mail will go on rotaion basis ,  for the first time when gravity form is submitted the mail goes to first email second time when it is submitted mail goes to second email address. When all email address limit is reached then an overflow email address is defined in settings and the email will be sent to the overflow email address. Also an admin email is set in the settings which will receive all the emails
